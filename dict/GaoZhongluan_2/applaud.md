# 音标

- 英 `/ə'plɔːd/`
- 美 `/ə'plɔd/`

# 词义

v. 赞同；称赞；向…喝彩
`to express strong approval of an idea, plan etc`



ap(表加强)+plaud(鼓掌)→鼓掌，喝彩

# 同义

vt. 赞同；称赞；向…喝采
`go along with` `be favorable to`

vi. 喝彩；鼓掌欢迎
`cheer for` `give it up`

# 例句

I applaud the decision to install more security cameras.
我极力赞成安装更多监控摄像机的决定。
