# 音标

- 英 `/nəʊ'teɪʃ(ə)n/`
- 美 `/no'teʃən/`

# 词义

n. (数学、音乐)的一套符号
`A system of notation is a set of written symbols that are used to represent something such as music or mathematics`

# 同义

n. [计][数]符号；乐谱；注释；记号法
`symbol` `notes` `sign` `mark`

# 同根

- adj. `notable` 值得注意的，显著的；著名的
- adj. `noted` 著名的；显著的；附有乐谱的
- n. `note` 笔记；音符；票据；注解；纸币；便笺；照会；调子
- n. `notable` 名人，显要人物
- v. `noted` 注意；记下（note的过去式和过去分词）
- vt. `note` 注意；记录；注解
- vt. `notate` 以符号表示

# 短语

- `musical notation` 乐谱；音乐符号
- `on board notation` 已装船批注
- `phonetic notation` 注音
- `numbered musical notation` 简谱
- `scientific notation` 科学记数法

# 例句

Musical notation was conceived for the C major scale and each line and space represents a note in this scale.
乐谱为C大音阶而构思，在这个音阶中的每一个线和间都代表一个音符。

...some other abstract notation system like a computer language.
…一些如计算机语言的其它抽象符号系统。
