# 音标

- 英 `/'festə/`
- 美 `/'fɛstɚ/`

# 词义

v. 脓疮，(怨恨等)郁积，恶化
`if an unpleasant feeling or problem festers, it gets worse because it has not been dealt with`

# 同义

vi. 溃烂；化脓
`gather head`

n. 溃烂；脓疮，脓疱
`ecthyma` `pustule`

# 例句

The dispute can be traced back to resentments which have festered for centuries.
这一争端可以追溯到积累了好几个世纪的仇怨。
