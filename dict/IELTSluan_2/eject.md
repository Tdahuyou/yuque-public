# 音标

- 英 `/ɪ'dʒekt/`
- 美 `/ɪ'dʒɛkt/`

# 词义

v. 驱逐，排出
`to make someone leave a place or building by using force`



e(出) + ject(扔) → 扔出来 → 驱逐， 逐出

# 同义

vt. 喷射；驱逐，逐出
`to spray` `spray on`

# 同根

- n. `ejector` [机] 喷射器；驱逐者；推出器；排出器
- n. `ejection` 喷出；排出物

# 例句

Officials used guard dogs to eject the protesters.
官员们用护卫犬驱走抗议者。
