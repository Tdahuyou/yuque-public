# 音标

- 英 `/ə'sɪdʒuəs/`
- 美 `/ə'sidjuəs/`

# 词义

adj. 勤勉的；专心的
`very careful to make sure that something is done properly or completely`



as + sid(坐) + uous → 坐得多的 → 勤勉的

# 同义

adj. 刻苦的，勤勉的
`diligent` `industrious`

# 同根

- adv. `assiduously` 刻苦地；勤勉地
- n. `assiduity` 勤勉；刻苦；殷勤

# 例句

Podulski had been assiduous in learning his adopted language.
珀杜斯基曾经勤奋地学习他所选的外语。
