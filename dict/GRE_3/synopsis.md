# 音标

- 英 `/sɪ'nɑːpsɪs/`
- 美 `/sɪ'nɑpsɪs/`

# 词义

n. 摘要，概要
`a short description of the main events or ideas in a book, film etc`



syn(一起)+op(看)+sis → 让大家一起看 → 摘要

# 同义

n. 概要，大纲
`program` `brief`

# 同根

- adj. `synoptic` 天气的；概要的；对观福音书的

# 例句

For each title there is a brief synopsis of the book.
每本书都附有一个该书的简短提要。
