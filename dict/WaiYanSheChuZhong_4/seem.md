# 音标

- 英 `/siːm/`
- 美 `/sim/`

# 词义

v. 看来；似乎
`if something seems to be true, there are qualities or facts that make people think it is true`

# 同义

vi. 似乎；像是；装作
`appear to`

# 同根

- adv. `seemingly` 看来似乎；表面上看来

# 短语

- `seem to be` 看来，好像
- `seem like` 看起来像；看上去像…；好像…
- `seem to do` 似乎做；好像

# 例句

Ann didn’t seem very sure.
安看来并不十分肯定。

It seems a foolish decision now.
现在看来这是一个愚蠢的决定。

It seems like you’re catching a cold, Taylor.
你好像感冒了，泰勒。

‘So Bill’s leaving her?’ ‘ So it seems (= that seems to be true ) .’
“这么说来，比尔要离开她？”“好像是这样。”
