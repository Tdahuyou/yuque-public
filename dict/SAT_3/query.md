# 音标

- 英 `/'kwiəri/`
- 美 `/'kwɪri/`

# 词义

v. 向…提问
`to express doubt about whether something is true or correct`

n. 疑问，问题
`a question that you ask to get information, or to check that something is true or correct`



quer (抱怨) + y → 为什么抱怨？大家都有疑问 (query)

# 同义

n. 疑问，质问；疑问号
`doubt` `question`

vt. [计]询问；对……表示疑问
`question` `inquire about`

vi. [计]询问；表示怀疑
`question` `inquire about`

# 短语

- `query language` 查询语言
- `query builder` 查询生成器
- `structured query language` 结构化查询语言
- `query the database` 查询数据库

# 例句

Both players queried the umpire’s decision.
两名球员都质疑裁判员的判罚。
