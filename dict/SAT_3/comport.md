# 音标

- 英 `/kəm'pɔːt/`
- 美 `/kəm'pɔ:t/`

# 词义

v. 举止
`to behave in a particular way`



com + port (带) → 一个人带有的全部仪态 → 举止

# 同义

vi. 相称；适合；一致
`gear` `cotton` `suit` `balance` `identify`

# 同根

- n. `comportment` 举止，态度；动作

# 短语

- `comport with` 适合，一致

# 例句

He comported himself in a way that won him respect.
他的行为为自己赢得了尊重。
