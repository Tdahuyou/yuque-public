# 音标

- 英 `/i'njuːməreit/`
- 美 `/ɪ'numəret/`

# 词义

v. 数，点；枚举，列举
`to name a list of things one by one`



e (出) + numer (数字) + ate → 列出数字 → 枚举， 列举

# 同义

vt. [计]列举；枚举；计算
`cast` `specify` `figure`

# 同根

- adj. `enumerable` 可列举的；可点数的
- n. `enumeration` 列举；[数] 计算；细目
- n. `enumerator` 人口调查员；计数器

# 例句

I enumerate the work that will have to be done.
我将必须要做的工作一一列举出来。
