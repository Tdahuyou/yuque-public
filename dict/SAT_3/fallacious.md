# 音标

- 英 `/fə'leiʃəs/`
- 美 `/fə'leʃəs/`

# 词义

adj. 谬误的；虚伪的；使人误解的
`containing or based on false ideas`



fall (欺骗) + aci + ous (充满了…) → 充满了欺骗 → 虚伪的

# 同义

adj. 谬误的；骗人的；靠不住的；不合理的
`uncertain` `unreasonable`

# 同根

- n. `fallacy` 谬论，谬误

# 例句

Such an argument is misleading, if not wholly fallacious.
这种论点即使不完全是谬误，也是误导人的。
