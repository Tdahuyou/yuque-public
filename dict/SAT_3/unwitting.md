# 音标

- 英 `/ʌn'witiŋ/`
- 美 `/ʌn'wɪtɪŋ/`

# 词义

adj. 不知情的




un (不) + wit (才智) + t + ing (…的) → 在这方面没有才智 → 不知情的

# 同义

adj. 不知情的；不知不觉的，无意的
`uninitiated` `involuntary`

# 同根

- adv. `unwittingly` 不知不觉地；不知情地；不经意地
