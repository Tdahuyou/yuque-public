# 音标

- 英 `/'pʌŋktjueit/`
- 美 `/ˈpʌŋktʃuet/`

# 词义

v. 加标点于；不时打断；强调
`to divide written work into sentences, phrases etc using  comma s  ,  full stop s  etc`



punct (刺) + uate → 锥刺股来提醒自己学习的重要性 → 强调

# 同义

vt. 不时打断；强调；加标点于

# 同根
