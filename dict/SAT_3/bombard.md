# 音标

- 英 `/bɔm'bɑːd/`
- 美 `/bɑm'bɑrd/`

# 词义

v. 炮轰；轰炸
`to attack a place for a long time using large weapons, bombs etc`



bomb (炸弹) + ard → 炮击， 以 (连珠炮式) 攻击某人 → 炮轰

# 同义

vt. 轰炸；炮击
`gun shot`

# 同根

- n. `bombardment` 轰炸；炮击
- n. `bomber` 轰炸机；投弹手

# 例句

I had been in action, bombarding the Normandy coast.
我参加过轰炸诺曼底海岸的行动。
