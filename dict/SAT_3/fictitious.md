# 音标

- 英 `/fik'tiʃəs/`
- 美 `/fɪk'tɪʃəs/`

# 词义

adj. 小说中的，假想的；假装的；虚伪的
`Fictitious is used to describe something that is false or does not exist, although some people claim that it is true or exists`



fict (做) + itious → 做出来的 → 做作的， 假的

# 同义

adj. 虚构的；[数]假想的；编造的；假装的
`affected` `imaginary` `assumed` `simulate`

# 同根

- adj. `fictional` 虚构的；小说的
- adj. `fictive` 虚构的；想象上的；虚伪的
- adv. `fictitiously` 虚构地；假地
- n. `fiction` 小说；虚构，编造；谎言

# 短语

- `fictitious capital` 虚拟资本；虚构资本
- `fictitious force` 假力；虚拟力；表观力

# 例句

We're interested in the source of these fictitious rumours.
我们对这些子虚乌有的谣言的来源感兴趣。
