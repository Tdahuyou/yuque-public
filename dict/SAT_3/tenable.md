# 音标

- 英 `/'tenəbl/`
- 美 `/undefined/`

# 词义

adj. 守得住的，可防守的；站得住脚的；有条理的
`a belief, argument etc that is tenable is reasonable and can be defended successfully`



ten (拿住) + able → 能够拿住的 → 站得住脚的

# 同义

adj. （主张等）站得住脚的；可维持的
`maintainable`

# 同根

- n. `tenability` 可防守；站得住脚

# 例句

an idea which is no longer tenable
不再成立的观点
