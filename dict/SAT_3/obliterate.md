# 音标

- 英 `/ə'blitəreit/`
- 美 `/ə'blɪtə'ret/`

# 词义

v. 消灭，冲刷；忘却；涂抹，遮蔽；擦去
`to remove a thought, feeling, or memory from someone’s mind`



ob (去掉) + liter (文字) + ate → 擦掉 (文字等)

# 同义

vt. 消灭；涂去；冲刷；忘掉
`destroy` `kill off`

# 同根

- n. `obliteration` 闭塞；涂去，删除；消灭
- n. `obliterator` 删除因子；混沌毁灭者

# 例句

Nothing could obliterate the memory of those tragic events.
那些悲惨事件的记忆是无法抹去的。
