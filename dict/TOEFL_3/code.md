# 音标

- 英 `/koud/`
- 美 `/kod/`

# 词义

vt. 编纂；把…编码
`to put a set of numbers, letters, or symbols on something to show what it is or give information about it`

n. 法典，法规；密码，代码
`a set of rules, laws, or principles that tell people how to behave`

# 同义

n. 代码，[遗][通信]密码；[计]编码；[法]法典
`password` `cryptogram`

vt. [计]编码；制成法典
`encode`

# 同根

- n. `coding` 译码
- n. `codification` 编纂，整理；法典编纂；法律汇编
- n. `coder` 编码器；编码员
- v. `coding` 把…编码（code的ing形式）
- vt. `codify` 编纂；将...编成法典；编成法典
