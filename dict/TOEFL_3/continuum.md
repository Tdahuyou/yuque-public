# 音标

- 英 `/kən'tɪnjuəm/`
- 美 `/kən'tɪnjuəm/`

# 词义

n. 统一体
`a scale of related things on which each one is only slightly different from the one before`

# 同根

- adj. `continuous` 连续的，持续的；继续的；连绵不断的
- adj. `continuing` 继续的；持续的；连续的
- adj. `continuant` 连续音的
- adv. `continuously` 连续不断地
- n. `continuity` 连续性；一连串；分镜头剧本
- n. `continuant` 连续音

# 短语

- `continuum mechanics` 连续介质力学
- `continuum theory` 连续介质理论
- `continuum damage mechanics` 连续损伤力学

# 例句

The Creole language is really various dialects arranged on a continuum.
克里奥 耳语其实是多种方言的一个统一体。

All the organisms in an ecosystem are part of an evolutionary continuum.
生态系统中的所有生物都是进化连续体的一个环节。
