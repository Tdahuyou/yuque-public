# 音标

- 英 `/'ʒɑːnrə/`
- 美 `/'ʒɑnrə/`

# 词义

n. 类型；流派
`a particular type of art, writing, music etc, which has certain features that all examples of this type share`



gen(种属) + re → 类型； 流派

# 同义

n. 类型；流派；风俗画
`description` `type` `style` `breed`

# 短语

- `genre painting` 风俗画（以日常生活为题材之写实画）；浮世绘

# 例句

a literary genre
文学体裁
