# 音标

- 英 `/'kwouʃnt/`
- 美 `/'kwoʃnt/`

# 词义

n. 商；份额
`the number which is obtained when one number is divided by another`



quot(多少) + ient → 有多少 → 商； 份额

# 同义

n. [数]商；系数；份额
`parameter` `coefficient` `lot` `Shang` `modulus`

# 短语

- `intelligence quotient` 智商（缩写为IQ）
- `intelligence quotient (IQ)` 智商
- `intelligence quotient (iq)` 智商
- `quotient space` 商空间
- `emotion quotient (EQ)` 情商
- `emotion quotient (eq)` 情商
- `emotional quotient` 情商；情感商数
- `difference quotient` 差商，均差；差分商
