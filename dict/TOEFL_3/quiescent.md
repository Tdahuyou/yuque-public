# 音标

- 英 `/kwi'esnt/`
- 美 `/kwɪ'ɛsnt/`

# 词义

adj. 静止的，寂静的；静态的
`not developing or doing anything, especially when this is only a temporary state`



qui(平静的) + esc + ent → 平静的状态 → 静止的

# 同义

adj. 静止的；不活动的；沉寂的
`still` `resting`

# 同根

- n. `quiescence` 静止；沉默
- n. `quiescency` 静止

# 例句

...a society which was politically quiescent and above all deferential.
...一个遵循顺从至上及政治上平静的社会。
