# 音标

- 英 `/ɪ'pɪtəmaɪz/`
- 美 `/ɪ'pɪtəmaɪz/`

# 词义

vt. 成为…的缩影；摘要，概括，缩写




epi(在…上) + tom(切) + ize → 剪切下来放在上面 → 成为…的缩影

# 同义

vt. 摘要；概括；成为…的缩影
`abstract` `brief`

# 同根

- n. `epitome` 缩影；摘要；象征
- vt. `epitomise` 象征；体现；代表（等于epitomize）

# 例句

This building epitomizes the spirit of the nineteenth century.
这栋建筑体现了19世纪的时代精神。
