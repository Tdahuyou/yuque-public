# 音标

- 英 `/ˌʌnɪnˈtenʃənəli/`
- 美 `/ˌʌnɪn'tenʃənəli/`

# 词义

adv. 无意地


# 同义

adv. 无意地；非故意地；非存心地
`inadvertently`

# 同根

- adj. `unintentional` 非故意的；无意识的

# 例句

I brushed up against him unintentionally.
我无意中碰了他一下。

But then there are the things he taught me unintentionally, like when he arrived home from work for the last time and crawled up the stairs.
但后来有些事情发生了，那些他无意中教我的事情，就如当他最后一次下班回家爬上楼梯的情景。

In fact, before I even start building the project, all unit tests are run to ensure that the integrity of the application code has not been unintentionally compromised.
实际上，在我每次开始构建项目之前，都将运行所有的单元测试，以确保应用程序代码的完整性没遭到无意破坏。
