# 音标

- 英 `/'deɪtəbeɪs/`
- 美 `/'detəbes/`

# 词义

n. 数据库，资料库
`a large amount of data stored in a computer system so that you can find and use it easily`



data(数据) + base(基地) → 数据的基地 → 数据库

# 同义

n. [计]数据库，资料库
`dbase` `informix`
