# 音标

- 英 `/dɪ'plɒɪ/`
- 美 `/dɪ'plɔɪ/`

# 词义

v. 配置；展开；使疏开


n. 部署


# 同义

vt. 配置；展开；使疏开
`station` `configurate`

vi. 部署；展开
`expand` `spread out`

n. 部署
`disposition`

# 同根

- n. `deployment` 调度，部署

# 例句

So far, however, we have said nothing about how to deploy it.
到现在，我们还没说任何如何部署的内容。

You can then customize and deploy these artifacts in the usual way.
然后，您可以以常规的方式定制和部署这些工件。

Now that you have seen the anatomy of each component, you can deploy them.
您已经了解了对每个组件的剖析，现在可以部署它们了。
