# 音标

- 英 `/krəʊm/`
- 美 `/krom/`

# 词义

n. 铬，铬合金；铬黄；谷歌浏览器
`a type of hard shiny metal`

# 同义

n. [化学]铬，铬合金；铬黄
`Paris yellow`

# 短语

- `chrome plating` 镀铬
- `chrome plated` 镀铬的
- `chrome ore` 铬矿；铬矿砂
- `chrome yellow` 铬黄
- `chrome steel` [冶]铬钢
- `chrome leather` 铬革
- `chrome finish` 镀铬
- `chrome oxide` 氧化铬；氧化铬绿
- `chrome plate` 镀铬压平板；铬掩模
- `hard chrome plating` n. 镀硬铬
- `chrome brick` 铬砖
- `hexavalent chrome` [机械]六价铬

# 例句

a chrome candleholder
铬合金蜡烛架
