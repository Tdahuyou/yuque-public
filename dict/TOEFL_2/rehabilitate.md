# 音标

- 英 `/riːə'bɪlɪteɪt/`
- 美 `/,riə'bɪlə,tet/`

# 词义

v. 修复,改造
`to improve a building or area so that it returns to the good condition it was in before`



re(再， 又) + hab(拥有) + ilit + ate → 重新拥有 → 使恢复原状， 修复

# 同义

vt. 使康复；使恢复名誉；使恢复原状
`repristinate` `unmake`

vi. 复兴；复权；恢复正常生活
`revive`

# 同根

- adj. `rehabilitative` 复职的，复原的
- n. `rehabilitation` 复原

# 例句

A lot of the older houses have now been rehabilitated.
很多老房子现在都进行了修葺。
