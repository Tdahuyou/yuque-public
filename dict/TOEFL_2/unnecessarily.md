# 同义

`needlessly` `redundantly`

# 同根

- adj. `unnecessary` 不必要的；多余的，无用的

# 例句

What you should not do is prevent or override it unnecessarily.
您不应该做的是不必要地阻止或重写它。

An awful lot of people with depression suffer unnecessarily because of a fear of using medication, but it can be very effective.
一个可怕的抑郁症的人很多遭受不必要的，因为使用药物的恐惧，但它可以是非常有效的。

This will only eat up memory unnecessarily.
这样只会不必要地消耗内存。
