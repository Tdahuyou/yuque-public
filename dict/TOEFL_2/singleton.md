# 音标

- 英 `/'sɪŋg(ə)lt(ə)n/`
- 美 `/'sɪŋɡltən/`

# 词义

n. 一个,独身


# 同义

n. 一个；独身
`yin` `celibacy`

# 同根

- n. `singleness` 单一；独身；真诚；专心

# 例句

Since there is only one singleton instance, any instance fields of a singleton will occur only once per class, just like static fields.
由于只有一个单实例，单例对象的域只会在一个类中出现一次，就像静态域一样。
