# 音标

- 英 `/əu'keɪ/`
- 美 `/undefined/`

# 词义

adv&int. 不错，很好；对吧


# 同义

adj. 好的；不错的
`good` `fine` `great` `bona`

adv. 行；很好
`certainly` `very good`

int. 好；可以
`okay` `zowie`

n. 同意
`agreement` `yes` `leave` `Okay` `consent`

# 短语

- `that's ok` 可以；那好吧
- `i'm ok` 我很好
- `everything is ok` 万事具备
- `ok button` OK按钮

# 例句

The plans have been okayed, so let’s get started as soon as possible.
计划已经获得批准，所以我们尽快开始吧。
