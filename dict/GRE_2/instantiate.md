# 音标

- 英 `/ɪn'stænʃɪeɪt/`
- 美 `/ɪns'tænʃɪet/`

# 词义

v. 例示
`to represent by an instance `

# 同义

vt. 例示，举例说明
`illustrate` `exemplify`

# 同根

- n. `instantiation` 实例化；[计] 例示

# 例句

You will never instantiate this class directly, but always create an instance of its subclass.
永远不要直接实例化这个类，而是创建它的子类的一个实例。
