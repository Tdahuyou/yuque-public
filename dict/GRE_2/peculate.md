# 音标

- 英 `/'pekjʊleɪt/`
- 美 `/'pekjuleit/`

# 词义

v. 挪用,盗用
`to appropriate or embezzle (public money) `



pecu(原义为“牛”， 引申为“钱财”) ＋ lat(搬运) + e → 把公有钱财搬回家里 → 挪用(公款)

# 同义

vt. 盗用，挪用（公款等）
`embezzle` `misappropriate`

# 同根

- n. `peculation` 侵吞公款；挪用公款；贪污
- n. `peculator` [法] 挪用公款者

# 例句

It means that we can modify and peculate the code.
这意味着我们可以盗用或修改其代码。

In an open network environment, it's hoped that the resources in the server is opened to the legal user in the network, and the server can prevent illegal user to peculate or demolish the resources;
在一个公开的网络环境，服务器上的资源希望只对网络中的合法用户开放，并能阻止非法用户进行盗用或破坏资源；
