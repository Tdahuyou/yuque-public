# 音标

- 英 `/'pleɪkeɪt/`
- 美 `/'pleket/`

# 词义

v. 安抚
`to make someone stop feeling angry`



plac(平静) + ate → 使平静 → 安抚

# 同义

vt. 抚慰；怀柔；使和解
`propitiate` `reconcile`

# 同根

- adj. `placable` 易抚慰的；易和解的；温和的
- adj. `placatory` 抚慰的；怀柔的；调解的
- n. `placation` 安抚

# 例句

These changes did little to placate the unions.
这些变革对于工会没有起到多少安抚作用。
