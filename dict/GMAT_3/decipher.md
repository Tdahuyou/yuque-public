# 音标

- 英 `/dɪ'saɪfər/`
- 美 `/dɪ'saɪfɚ/`

# 词义

v. 破译
`to change a message written in a  code  into ordinary language so that you can read it`



de(去掉) + cipher(密码) → 破译

# 同根

- adj. `decipherable` 可判读的；可解释的
- n. `decipherer` 译码器；译码员
- n. `decipherment` 解读；翻译；[通信] 解密

# 例句

I'm still no closer to deciphering the code.
破译这个密码我还是没有进展。
