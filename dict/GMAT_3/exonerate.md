# 音标

- 英 `/ɪg'zɑːnəreɪt/`
- 美 `/ɪɡ'zɑnəret/`

# 词义

v. 证明…无罪；免受指控
`to state officially that someone who has been blamed for something is not guilty`



ex(出) + oner(负担) + ate → 走出负担 → 证明…无罪

# 同根

- adj. `exonerative` 免罪的；释罪的；免除的
- n. `exoneration` 免除；免罪

# 例句

The official report basically exonerated everyone.
该官方报告基本洗脱了所有人的干系。
