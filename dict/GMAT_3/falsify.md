# 音标

- 英 `/'fɔːlsɪfaɪ/`
- 美 `/'fɔlsɪfaɪ/`

# 词义

v. 篡改；说谎
`to change figures, records etc so that they contain false information`



fals(假) + ify → 造假 → 篡改

# 同义

vt. 伪造；篡改；歪曲；证明...虚假
`colour` `doctor`

vi. 撒谎
`tell a lie`

# 同根

- adj. `false` 错误的；虚伪的；伪造的
- adv. `false` 欺诈地
- n. `falsification` 伪造；歪曲；[审计] 篡改；证明是假
- n. `falsifier` [法] 伪造者；伪证的人；撒谎的人

# 例句

The file was altered to falsify the evidence.
文件被动了手脚，以此伪造证据。
