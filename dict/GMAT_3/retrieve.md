# 音标

- 英 `/rɪ'triːv/`
- 美 `/rɪ'triv/`

# 词义

v. 取回；恢复；检索
`to get back information that has been stored in the memory of a computer`



re + trieve(=find找到) → 重新找到 → 取回

# 同义

vt. [计]检索；恢复；重新得到
`restore` `repair`

n. [计]检索；恢复，取回
`recovery` `restoration` `renewal`

# 同根

- adj. `retrievable` 可取回的；可补偿的；可检索的
- n. `retrieval` 检索；恢复；取回；拯救
- n. `retriever` （训练成能寻回猎物的）猎犬；取回的人

# 短语

- `retrieve data` 检索数据
- `beyond retrieve` 不可恢复；不可挽回

# 例句

The new version of the software automatically retrieves digital information.
新版软件会自动检索数字信息。
