# 音标

- 英 `/ ˌriː'waɪnd /`
- 美 `/,ri'waɪnd/`

# 词义

v. 重绕；回转
`to make a  cassette  tape or  video  go backwards in order to see or hear it again`



re(重新)+wind(缠绕)→重绕

# 例句

Would you please rewind the thread onto the reel?
请你把这些线重新绕在线轴儿上好吗?。

All the things I would change if we could only rewind.
我愿改变一切，如果我们可以重回往昔。
