# 音标

- 英 `/ɪn'terəgeɪt/`
- 美 `/ɪn'tɛrəɡet/`

# 词义

v. 审问；质问；[计]询问
`to ask someone a lot of questions for a long time in order to get information, sometimes using threats`



inter (相互) + rog (问) + ate (使) → 有问有答 → 询问

# 同义

vt. 审问；质问；[计]询问
`question` `query`

vi. 审问；质问
`question` `bring to account`

# 同根

- adj. `interrogative` 疑问的；质问的
- adj. `interrogatory` 质问的；疑问的
- n. `interrogation` 讯问；审问；[语] 疑问句
- n. `interrogative` 疑问词
- n. `interrogator` 质问者；询问机
- n. `interrogatory` 质问；疑问

# 例句

The police interrogated the suspect for several hours.
警察盘问了这名嫌疑犯好几个小时。
