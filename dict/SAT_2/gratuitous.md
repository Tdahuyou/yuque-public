# 音标

- 英 `/grə'tjuːɪtəs/`
- 美 `/ɡrə'tuɪtəs/`

# 词义

adj. 无理由的，无端的；免费的
`said or done without a good reason, in a way that offends someone`



grat (高兴) + uit + ous (…的) → 无偿获助， 当然高兴 → 无偿的

# 同义

adj. 无理由的，无端的；免费的
`free` `groundless`

# 同根

- adj. `gratis` 免费的
- adv. `gratis` 免费地
- adv. `gratuitously` 无偿地；不必要地；无缘无故地；平白地

# 例句

children’s books which include gratuitous violence
包含无谓暴力内容的儿童书籍
