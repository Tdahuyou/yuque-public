# 音标

- 英 `/'fjuːmɪgeɪt/`
- 美 `/'fjumɪɡet/`

# 词义

v. 熏制；香薰；用烟熏消毒
`to remove disease,  bacteria  , insects etc from somewhere using chemicals, smoke, or gas`



fum ( = fume， 烟) + igate (用…的) → 用烟消毒 → 熏蒸 (消毒)

# 同根

- n. `fumigation` 烟熏法；熏烟消毒法
- n. `fumigant` 熏剂；熏蒸消毒剂
- n. `fumigator` 熏蒸消毒器；烟熏器

# 例句

...fruit which has been treated with insecticide and fumigated.
...杀过虫、消过毒的水果。
