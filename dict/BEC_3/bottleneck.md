# 音标

- 英 `/'bɔtlnek/`
- 美 `/'bɑtlnɛk/`

# 词义

n. 瓶颈，障碍
`a place in a road where the traffic cannot pass easily, so that there are a lot of delays`



bottle(瓶子) + neck(脖子， 颈部) → 瓶颈

# 同义

n. [包装]瓶颈；障碍物
`barrier` `block`

# 短语

- `bottleneck problem` 瓶颈问题

# 例句

Understaffing has caused a real bottleneck.
人手不足严重妨碍了整个进度。
