# 音标

- 英 `/'retrəuspekt/`
- 美 `/'rɛtrəspɛkt/`

# 词义

n. 回顾
`thinking back to a time in the past, especially with the advantage of knowing more now than you did then`



retro(往后) + spect(看见) → 回过头往后看 → 回顾

# 同义

n. 回顾，追溯
`review` `recalling`

vi. 回顾，追溯；回想
`to look back` `look back on`

vt. 回顾；追忆
`to look back` `look back on`

# 同根

- adj. `retrospective` 回顾的；怀旧的；可追溯的
- adv. `retrospectively` 回顾地
- n. `retrospective` 回顾展
- n. `retrospection` 回顾；反省；追忆

# 短语

- `in retrospect` 回顾；回顾往事；检讨过去

# 例句

In retrospect, I wonder if we should have done more.
回过头看，我在想我们是否应该做得更多些。
