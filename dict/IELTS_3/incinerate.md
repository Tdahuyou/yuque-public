# 音标

- 英 `/ɪn'sɪnəreɪt/`
- 美 `/in'sinəreit/`

# 词义

vt. 焚化，焚毁，把…烧成灰烬
`to burn something completely in order to destroy it`



in(进入) + ciner(灰) + ate(使…) → 使成灰 → 焚化， 焚毁

# 同义

vt. 焚化；烧成灰
`cremate`

# 同根

- n. `incineration` 焚化；烧成灰

# 例句

All the infected clothing was incinerated.
所有沾染了细菌的衣服都被焚化了。
