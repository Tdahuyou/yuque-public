# 音标

- 英 `/ə'baund/`
- 美 `/ə'baʊnd/`

# 词义

vi. 富于；充满
`If things abound, or if a place abounds with things, there are very large numbers of them`



a(一个) + bound(界限， 范围) → 所有物品都在一个范围内 → 富于； 充满

# 同义

vi. 富于；充满
`teem with` `be imbued with`

# 同根

- adj. `abounding` 丰富的；大量的
- v. `abounding` 大量存在；盛产（abound的ing形式）

# 短语

- `abound in` vt. 富于
- `abound with` 充满

# 例句

Stories abound about when he was in charge.
他负责时传闻很多。
