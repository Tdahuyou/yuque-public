# 音标

- 英 `/ɪn'vaɪələbl/`
- 美 `/ɪn'vaɪələbl/`

# 词义

adj. 不可侵犯的，不容亵渎的，不容破坏的
`an inviolable right, law, principle etc is extremely important and should be treated with respect and not broken or removed`



in(不) + viol(违反， 冒犯) + able(能…的) → 不可侵犯的

# 同义

adj. 不可侵犯的；神圣的；不可亵渎的
`sacred` `blessed`

# 同根

- adj. `inviolate` 无污点的；未亵渎的，未受侵犯的

# 短语

- `sacred and inviolable` 神圣不可侵犯

# 例句

The game had a single inviolable rule: obstacles were to be overcome, not circumvented.
这个游戏有一个单一的不可违反的规则：困难是被克服的，不是被回避的。
