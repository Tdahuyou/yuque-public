# 音标

- 英 `/ɪn'ækjʊrət/`
- 美 `/ɪn'ækjərət/`

# 词义

adj. 错误的




in(非) + accurate(精确的) → 非精确的 → 错误的

# 同义

adj. 错误的
`false` `wrong` `incorrect` `improper` `erroneous`

# 同根

- adv. `inaccurately` 不准确地；不正确地
- n. `inaccuracy` 错误；不精确

# 例句

This is a silly and inaccurate way to think about time, however.
然而，这是一种愚蠢和错误的看待时间的方法。

It could also be an invalidation notice specifying that part of data should not be considered in the integration process because it is inaccurate.
它经常是一个无效的通告，部分的数据集成过程中应该不会被考虑，因为他们是错误的。
