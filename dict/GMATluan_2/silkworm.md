# 音标

- 英 `/'sɪlkwɜːm/`
- 美 `/'sɪlkwɝm/`

# 词义

n. 蚕，桑蚕
`a type of  caterpillar  which produces silk thread`



silk(蚕丝) + worm(虫) → 吐蚕丝的虫 → 蚕

# 短语

- `silkworm chrysalis` 蚕蛹
- `silkworm moth` 蚕蛾
- `silkworm pupa` 蚕蛹

# 例句

The first silkworm diseases began to appear in 1845, creating an epidemic.
1845年，首批桑蚕疾病开始出现，并引起了一次流行。
