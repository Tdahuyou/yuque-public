# 音标

- 英 `/seɪm/`
- 美 `/sem/`

# 词义

adj. 相同的
`the same person, place, thing etc is one particular person etc and not a different one`

pron. 同样
`used to say that two or more people or things are exactly like each other`

adv. 同样地
`in the same way`

# 同义

adj. 相同的；同一的；上述的（通常与the连用）；无变化的
`said` `uniform` `self` `identical` `parallel`

adv. 同样地（通常与the连用）
`equally` `similarly` `as` `likewise` `homogeneously`

# 同根

- n. `sameness` 相同；千篇一律；单调

# 短语

- `same time` 同时
- `same as` 等同于；与…一致
- `same way` 同样；同样的方法；同样的感受
- `in the same time` 在同一时间
- `same type` 同型
- `same place` 老地方；同名点
- `all the same` 仍然
- `much the same` 几乎相同，大致一样
- `just the same` 还是；完全一样；照旧（等于all the same）
- `one and the same` 同一个；完全一回事
- `as same as` 和…一样
- `at the same level` 本级；在同一水平上
- `for the same reason` 为了同样的理由
- `same order` 同阶；同序
- `stay the same` 保持不变；保持一致
- `by the same token` 同样地；出于同样原因
- `in the same manner` 同样地；以同样的方式
- `look the same` 看起来一样
- `the very same` 完全相同的；正是这个
- `same to you` 彼此彼此；我也这样祝愿你

# 例句

He sits in the same chair every night.
他每天晚上都坐在同一把椅子上。

The coins may look the same but one’s a forgery.
这些硬币虽然看起来一样，但其中有一个是假币。

Everyone had to dress the same as a well-known historical figure.
每个人都必须装扮成一位历史名人。
