# 音标

- 英 `/prɪ'skraɪb/`
- 美 `/prɪ'skraɪb/`

# 词义

v. 指示,规定；处(方),开(药)
`to state officially what should be done in a particular situation`



pre(预先)+scrib(写)+e→预先写好药方→开处方

# 同义

vi. 规定；开药方
`to stipulate` `to formulate`

vt. 规定；开处方
`define` `regulate` `state` `rule`

# 同根

- adj. `prescribed` 规定的

# 短语

- `prescribe for` 开处方，开药方

# 例句

What punishment does the law prescribe for this crime?
法律规定这种罪行应受何处罚？

If these don’t work I may have to prescribe you something stronger.
如果这些药无效，我也许得给你开一些药效更强的。
